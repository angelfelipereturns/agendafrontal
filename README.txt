-------------------------
INFORMACION DE DESARROLLO
-------------------------
Aplicación desarrollada para probar dos proyectos Spring Boot que se comunican 
para realizar una tarea común en una aplicación de agenda de contactos.
Esta es la parte de la aplicación que se encarga de mostrar el frontal de la
agenda de contactos en la web de usuario, con las acciones de consultas, 
actualizaciones y borrados, consumiendo para ello una API Restful.
